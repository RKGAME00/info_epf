JavaFX build instructions (Windows Powershell)

Simple non-modular compile/run using JavaFX SDK:

1. Download/OpenJFX SDK matching your JDK and extract. Set PATH_TO_FX to the 'lib' folder.

2. Compile:
   javac --module-path $env:PATH_TO_FX --add-modules javafx.controls,javafx.fxml -d out src/gui_javafx/*.java

3. Run:
   java --module-path $env:PATH_TO_FX --add-modules javafx.controls,javafx.fxml -cp out gui_javafx.Main

If you use Maven/Gradle it's easier; create a JavaFX project template and add these sources.
