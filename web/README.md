# Projets IUT - HTML CSS JavaScript

## Présentation du dépôt

Ce dépôt contient l'ensemble de mes projets réalisés à l'IUT. Vous y retrouverez les deux TP à rendre du cours HTML CSS JavaScript ainsi que mon portfolio personnel.

## Les travaux rendus

### TP1 - CV en ligne

J'ai créé un CV en ligne responsive qui présente mon parcours, mes compétences et mes expériences. Le design est sobre et épuré, sans fioritures inutiles. Le site s'adapte aux différentes tailles d'écran : ordinateur, tablette et smartphone.

Tout a été codé en HTML et CSS natif, sans utiliser aucun framework conformément aux consignes.

### TP2 - Portes logiques

J'ai développé une page interactive pour comprendre le fonctionnement des portes logiques : ET, OU, XOR, NAND, NOR et NOT. L'utilisateur peut cliquer sur les boutons pour changer les valeurs des entrées et voir directement le résultat en sortie.

Chaque porte est accompagnée d'une explication et d'une table de vérité. Des emplacements sont prévus pour insérer les images des symboles. Toute l'interaction est gérée en JavaScript natif, avec beaucoup de messages dans la console pour suivre le code.

Là encore, aucun framework ni librairie n'a été utilisé.

## Mes réflexions sur ces TP

J'aurais aimé aller beaucoup plus loin dans ces réalisations. Mon idée de départ était de créer un site avec des animations poussées sur le scroll, comme on peut en voir sur les pages produits Apple (iPhone, AirPods). Ces sites proposent des animations 3D qui tournent et des transitions très fluides au lieu d'un simple défilement.

Pour des raisons personnelles, je n'ai pas pu travailler sur ces projets de manière aussi poussée que souhaité. J'ai donc opté pour quelque chose de sobre, à l'image de mon CV. J'ai tout de même essayé d'utiliser un maximum de techniques vues en cours, en respectant le cadre sans framework ni librairie.

## Mon portfolio

J'avais déjà commencé à travailler sur mon portfolio l'année dernière dans le cadre de mon BUT. Vous le retrouverez également dans ce dépôt.

### Projet de refonte

J'aimerais faire un rework complet de ce portfolio pour le moderniser et y intégrer de nouvelles fonctionnalités. Mon objectif est d'apprendre en utilisant des outils et des techniques modernes, même si certains peuvent sembler "overkill" pour un simple portfolio.

Par exemple, je fais de la musique et j'aimerais intégrer un lecteur multimédia personnalisé pour partager mes compositions.

### Mes questions

J'aurais besoin de conseils sur comment aborder cette refonte de manière structurée. Quels frameworks utiliser ? Comment organiser le projet ? Quelles sont les bonnes pratiques pour intégrer des fonctionnalités avancées comme un lecteur audio, des animations scroll, ou des connexions API ?

**Le point important : je ne veux pas perdre ce que j'ai déjà fait.** Comment faire évoluer mon portfolio existant sans tout recommencer de zéro ?

J'aimerais avoir votre avis sur les outils à utiliser et la bonne manière d'aborder tout ça pour avancer de façon claire et méthodique.

## Conclusion

Ce dépôt représente mon travail dans le cadre du cours, qui respecte les contraintes imposées, ainsi que le début d'une réflexion sur mon apprentissage du développement web.

Je suis ouvert à vos conseils et retours, que ce soit sur le code produit ou sur la direction à prendre pour faire évoluer mon portfolio.
